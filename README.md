# 📌 About The Project

CODesta Banking System is a desktop application that simulates real banking operations in a simple and user-friendly way.

The project was developed as part of an academic project at **Mansoura University**.

It allows users to manage accounts, perform transactions, and track financial activities using Object-Oriented Programming principles.

# 🧠 OOP Concepts Used

## 🔒 Encapsulation
Protecting sensitive account data.

## 👨‍👦 Inheritance

```plaintext
Account
├── SavingAccount
└── CheckingAccount
```

## 🔄 Polymorphism
Using virtual methods:
- Deposit()
- Withdraw()

## 🎭 Abstraction
Using abstract class:
- Account
